## Quick Reference

### File Categories
- **Backend:** API, services, database
- **Frontend:** Components, pages, styles
- **Infrastructure:** Docker, CI/CD, config
- **Tests:** Unit, integration, E2E
- **Docs:** Documentation updates

### Severity Levels
- **HIGH:** Blocks progress or causes failures
- **MEDIUM:** Impacts functionality but has workaround
- **LOW:** Minor issue or cosmetic

### Priority Levels
- **HIGH:** Must be done next session
- **MEDIUM:** Should be done soon
- **LOW:** Nice to have when time permits
