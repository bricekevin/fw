Install priceDisplay

Flash a fresh M5Stack Core in about a minute — no toolchain, no source clone. A crypto & stock price ticker with candlestick charts.

  1. Plug your M5Stack Core into a USB port on this computer (data cable, not charge-only).
  2. Click Connect below and pick the device from the list (its name looks like cu.usbserial-… or cu.SLAB_USBtoUART).
  3. Click Install priceDisplay on the panel that appears, confirm the erase prompt, and wait. The first install writes the bootloader, firmware, and the chart images (~1 min).
  4. Insert an SD card with a ccticker.cfg file (WiFi credentials, trading pairs, timezone). When the device reboots it reads that config; if it can't find WiFi it opens the SETUP PRICE TICKER soft-AP — join it from your phone to enter WiFi.
Browser: works in Chrome or Edge on desktop (they have WebSerial). Safari and Firefox don't yet — open this page in Chrome.
After the first install the device updates itself over WiFi — every six hours it checks for a newer release and rolls it in automatically. You only need this page for a brand-new device or a recovery flash.
Firmware artifacts: github.com/bricekevin/fw/releases.
