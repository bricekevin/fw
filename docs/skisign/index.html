Install SD Coaster Sign

Flash the San Diego COASTER LED map sign in about a minute — no toolchain, no source clone. The sign shows live trains straight from the transit feed; it only needs Wi-Fi and power.

  1. Plug the sign's board into a USB port on this computer (data cable, not charge-only).
  2. Click Connect below and pick the device from the list (its name looks like cu.usbserial-… or CH340).
  3. Click Install SD Coaster Sign on the panel that appears, confirm the erase prompt, and wait (~1 min).
  4. When the board reboots, join the Wi-Fi network SETUP SD COASTER from your phone (password 123456789) and enter your home Wi-Fi. That's it — trains appear within a minute.
Browser: works in Chrome or Edge on desktop (they have WebSerial). Safari and Firefox don't yet — open this page in Chrome.
After the first install the sign updates itself over Wi-Fi — every six hours it checks for a newer release and rolls it in automatically. You only need this page for a brand-new board or a recovery flash.
