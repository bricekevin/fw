Install Social Meter

Flash a fresh M5Stack Core Basic in about a minute — no toolchain, no source clone.

  1. Plug your M5Stack Core Basic into a USB-C port on this computer (data cable, not charge-only).
  2. Click Connect below and pick the device from the list (its name looks like cu.usbserial-… or cu.SLAB_USBtoUART).
  3. Click Install Social Meter on the panel that appears, confirm the erase prompt, and wait. The first install writes the bootloader and the firmware (~1 min).
  4. When the device reboots, follow the on-screen setup (STEP 1 of 2 → join the soft-AP Social Meter-XXXXXX on your phone, then STEP 2 of 2 opens the setup page where you pick what to count — a platform, your handle, and a key only if that platform needs one).
Browser: works in Chrome or Edge on desktop (they have WebSerial). Safari and Firefox don't yet — open this page in Chrome.
After the first install the device updates itself over WiFi — every six hours it checks for a newer release and rolls it in automatically. You only need this page for a brand-new device or a recovery flash.
Firmware artifacts: github.com/bricekevin/fw/releases.
